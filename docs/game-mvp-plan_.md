현재 프로젝트에 “10분 용병단” 웹 미니게임 MVP를 구현해라.

목표는 예쁜 완성판이 아니라, 실제로 브라우저에서 실행되고 플레이 가능한 MVP다.
기획서와 일치하는 핵심 루프를 우선 구현한다.

게임 제목:
10분 용병단

한 줄 설명:
플레이어가 직접 공격하지 않고, 자동으로 싸우는 용병들을 고용·강화하며 10분 동안 살아남는 웹 기반 자동전투 생존 미니게임.

중요:
Phaser 버전은 package.json에 설치된 버전을 기준으로 구현해라.
만약 새로 설치해야 한다면 안정성과 예제 호환성을 우선하여 Phaser 3를 사용해라.
Phaser 4를 사용한다면 Phaser 3 API를 섞지 말고, 설치된 타입 정의를 확인해서 타입 에러 없이 구현해라.

실행 목표:

npm install
npm run dev
위 명령으로 로컬에서 바로 실행되어야 한다.

작업 전 확인:

현재 프로젝트 구조를 먼저 확인해라.
package.json이 있는지 확인해라.
이미 React/Vite 프로젝트가 있으면 그 구조를 유지해라.
프로젝트가 비어 있거나 없다면 React + Vite + TypeScript 프로젝트 구조를 생성해라.
기존 파일을 무분별하게 삭제하지 마라.
구현 후 npm run build 또는 npm run typecheck가 가능하면 실행해서 타입/빌드 오류를 고쳐라.
